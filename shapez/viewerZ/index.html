About

With this tool you can generate any shape supported in shapez.io! This can be useful to analyze shapes or draft new ones.

Instructions

Shapes can be generated with a such called short key. Each shape consists of up to 4 layers, and each layers consists of 4 quadrants.


Each quadrant is described via two letters - At first, an uppercase letter describing the shape, then followed by a lowercase letter describing its color. If the quadrant is empty, -- is used


Layers itself are then seperated via :, starting with the lowest layer.
